Développement d'un outil de récupération de données sismiques

Développement logiciel - Python - requêtes web
Electronicien Science à l'Institut Polaire Français
2022

J'ai réalisé l'analyse quotidienne des ondes sismiques meusrées sur la station sismique locale. Afin de gagner en efficacité dans la comparaison de mes analyse avec la détection de séismes mondiale, j'ai développé un outil pour interroger les bases de données sismiques avec les information pertiantes.

L'outil est en open-source et est disponible sur ma page GitHub :
https://github.com/IbanAino/Seisms
